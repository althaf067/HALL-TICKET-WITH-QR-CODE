🎓 Hall Ticket Generation System (QR Enabled)

A fully functional Hall Ticket Generator Web App that allows users to create, manage, and print examination hall tickets with QR code verification, student photo upload, and an admin dashboard — all in a single HTML file.

🚀 Features:

🧾**Hall Ticket Generation**
• Enter student & exam details
• Add multiple subjects with date & time
• Auto-generate professional hall ticket
• Unique Ticket ID for each entry

📷 **Photo Upload**
• Upload student photograph
• Preview before generating ticket
• Embedded directly into hall ticket

🔳 **QR Code Integration**
• Automatically generates QR code
• Encodes student & exam details
• Useful for verification & validation

🖨️ **Print Ready Design**
• Clean A4 print layout
• Print directly from browser
• Styled for official use

📊 **Admin Dashboard**
• View all generated tickets
• Search & filter by:
      •Name
      •Roll number
      •Department
      •Semester
• Statistics:
      •Total tickets
      •Departments
      •Semesters
      •Last generated record

💾 **Local Storage System**
• Stores records in memory (no backend required)
• Fast and lightweight

🛠️ **Technologies Used**
• HTML5
• CSS3 (Advanced UI Styling)
• Vanilla JavaScript
• Custom-built QR Code Generator

📂 **Project Structure**
📁 hall-ticket-generator
 └── hall ticket with qr code.html

This project is fully self-contained in a single HTML file.
